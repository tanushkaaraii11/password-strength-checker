# Password Strength Checker

A simple Python program that checks whether a password is **Strong**, **Medium**, or **Weak** based on common password security rules.

## Features

- Checks password length (minimum 8 characters)
- Detects uppercase letters
- Detects lowercase letters
- Detects numbers
- Detects special characters
- Classifies passwords as:
  -  Strong
  -  Medium
  -  Weak
- Provides personalized suggestions to improve weak and medium passwords

## Technologies Used

- Python 3

## How to Run

1. Download the project.
2. Open a terminal in the project folder.
3. Run:

```bash
python password_strength_checker.py
```
